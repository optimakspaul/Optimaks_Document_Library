## Policy

Optimaks should clearly state what the client receives:

1. Website access
2. Admin access
3. Exported data
4. Static files, if applicable
5. Repository access, if agreed
6. Documentation and handover notes

## Recommended Position

For productized systems, Optimaks should not automatically transfer the full reusable core system source code to each client unless explicitly agreed and priced.

## Demo Policy

Demo landing pages can be shared for sales purposes. Full automation module code should not be handed over casually because it contains reusable product logic.
