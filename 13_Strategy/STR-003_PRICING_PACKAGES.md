## 1. Pricing Principle

Optimaks should not compete only on being cheaper than PSG vendors or web agencies.

Optimaks should price around:

```text
Speed to value + practical workflow + productized delivery + low ERP complexity.
```

---

## 2. Package 1 — Starter: Lead Capture Website

```text
Setup: S$1,500
Monthly: S$99
```

Best for:

```text
- company has weak or no landing page
- receives WhatsApp enquiries manually
- wants better enquiry capture before workflow automation
```

Includes:

```text
- one-page landing page
- WhatsApp CTA
- lead capture form
- basic enquiry dashboard
- Google Maps / review section
- mobile optimization
- basic monthly maintenance
```

Excludes:

```text
- job card
- invoice tracker
- complex CRM
- custom workflow automation
```

---

## 3. Package 2 — Growth: Lead to Booking Workflow

```text
Setup: S$2,800
Monthly: S$199
```

Best for:

```text
- company receives steady enquiries
- admin uses WhatsApp / Excel to follow up
- quotes and bookings are not consistently tracked
```

Includes:

```text
- all Starter features
- customer profile
- quote tracking
- booking schedule
- lead status pipeline
- WhatsApp message templates
- basic reporting
```

Recommended as main package.

---

## 4. Package 3 — Pro: Service Workflow OS

```text
Setup: S$3,800 - S$4,800
Monthly: S$299 - S$399
```

Best for:

```text
- 3-10 technicians / workers
- needs job completion tracking
- needs invoice status and maintenance reminder
```

Includes:

```text
- all Growth features
- job card
- technician task view
- completion photo upload
- invoice status tracker
- maintenance reminder
- monthly performance dashboard
- priority support
```

---

## 5. Founding Client Offer

For early validation, Optimaks may use:

```text
Founding Client Offer
S$1,999 setup + S$149/month
Limited to first 3 clients
```

This should be positioned as a limited early client package, not as permanent public pricing.

---

## 6. Diagnostic Offer

Optional sales entry:

```text
S$800 7-Day Lead Capture Diagnostic
Can be offset against Growth or Pro setup fee.
```

This should be used as a sales diagnostic, not as the main advertised package.

---

## 7. Pricing Guardrails

Avoid:

```text
- unlimited custom requests
- performance guarantees without tracking data
- public underpricing that traps future delivery margin
- one-off website-only work that weakens workflow positioning
```

Require:

```text
- clear included features
- clear excluded features
- upgrade path
- monthly value report
- support boundary
```
