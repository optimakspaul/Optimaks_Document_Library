| Purpose | Checklist for PDPA and basic data protection. |

---


- [ ] Only necessary data collected.
- [ ] Purpose of collection clear.
- [ ] Privacy/PDPA notice prepared if public-facing.
- [ ] Access controlled by role.
- [ ] No secrets in frontend.
- [ ] No sensitive customer data in public repo.
- [ ] Third-party tools listed.
- [ ] Data export/handover method considered.
- [ ] Deletion/correction process considered.
