# CHK-MKT-001 Short Video Delivery Checklist

| Field | Value |
|---|---|
| Document Code | CHK-MKT-001 |
| Version | v1.5.7 Short Video Lead Kit Partner Governance Patch |
| Effective Date | 2026-05-16 |
| Owner | Optimaks Pte Ltd |
| Status | Active Checklist |
| Purpose | Ensure every Short Video Lead Kit delivery follows approved strategy, CTA, vendor and workflow controls. |

---

## 1. Pre-Sales Check

- [ ] Client understands this is not generic social media management.
- [ ] Client understands short video is connected to WhatsApp / workflow tracking.
- [ ] Client understands no views / leads / sales are guaranteed.
- [ ] Correct package selected: Starter / Growth / Pro.
- [ ] Client responsibilities explained.

---

## 2. Scope Check

- [ ] Number of videos confirmed.
- [ ] Filming or editing-only model confirmed.
- [ ] Revision limit confirmed.
- [ ] Delivery timeline confirmed.
- [ ] Posting responsibility confirmed.
- [ ] Paid ads excluded unless separately scoped.

---

## 3. Vendor Check

- [ ] Vendor approved in evaluation matrix.
- [ ] Vendor accepts white-label / hybrid terms if applicable.
- [ ] No watermark requirement confirmed.
- [ ] Turnaround time confirmed.
- [ ] Revision policy confirmed.
- [ ] Client non-solicitation confirmed.

---

## 4. Content Strategy Check

- [ ] Monthly themes approved.
- [ ] Hook approved.
- [ ] Script direction approved.
- [ ] CTA approved.
- [ ] Landing page / WhatsApp destination confirmed.
- [ ] Video-to-lead tracking field prepared.

---

## 5. Filming / Material Check

### For Starter

- [ ] Client shooting checklist sent.
- [ ] Footage deadline confirmed.
- [ ] File upload method confirmed.
- [ ] Required before / after shots listed.

### For Growth / Pro

- [ ] Filming date confirmed.
- [ ] Site access confirmed.
- [ ] Homeowner / tenant consent confirmed.
- [ ] Technician / staff participation confirmed.
- [ ] Filming duration limit confirmed.

---

## 6. Delivery QA

- [ ] Vertical format delivered.
- [ ] Subtitles included.
- [ ] CTA visible or stated.
- [ ] No unauthorized vendor watermark.
- [ ] No false claims.
- [ ] No privacy-sensitive information exposed.
- [ ] Video matches approved service scenario.
- [ ] File names and version numbers are clear.

---

## 7. Workflow Tracking Check

- [ ] Lead source tracking configured.
- [ ] WhatsApp link tested.
- [ ] Landing page link tested.
- [ ] Enquiry appears in lead dashboard.
- [ ] Campaign / video source field tested.
- [ ] Quote / booking path confirmed.

---

## 8. Monthly Review Check

- [ ] Number of videos delivered.
- [ ] Number of enquiries recorded.
- [ ] Quote / booking conversion reviewed.
- [ ] Client feedback recorded.
- [ ] Vendor performance reviewed.
- [ ] Next month themes drafted.
