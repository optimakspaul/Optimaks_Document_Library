# CHK-DATA-001 Mock Demo Data Checklist

Use before demo, preview, staging or production.

## Mock Data

- [ ] Fake names used.
- [ ] Fake emails used.
- [ ] Fake phone numbers used.
- [ ] Fake addresses used.
- [ ] Mock data stored in approved location.

## Demo Data

- [ ] Clearly labeled as DEMO.
- [ ] No unauthorized real company logo.
- [ ] No real customer information.
- [ ] No real invoice/quote content.
- [ ] No production seed contamination.

## Client Data

- [ ] Real client data not pasted into AI prompts.
- [ ] Sensitive client info redacted.
- [ ] Client logo usage authorized if public.
- [ ] Demo environment separated from production.

## Final Check

- [ ] Screenshots safe to share.
- [ ] Preview URL safe to show.
- [ ] No private operational data visible.
