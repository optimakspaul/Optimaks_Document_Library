# Optimaks Document Library v1.5.1 WAS Positioning Patch

This repository is the official GitHub-ready document library for Optimaks development governance.

本文件庫是 Optimaks 的正式開發治理文件庫。v1.5.1 是在 v1.5 Cleanup + Fusion 的基礎上，新增「產品定位 / Workflow Automation System / SME Operating OS」憲法級原則。

## What v1.5.1 Adds

v1.5.1 does not replace v1.5. It adds a formal product positioning layer to clarify what Optimaks builds and what it does not build.

The key principle is:

> Optimaks does not build generic SaaS products. Optimaks builds industry-specific Workflow Automation Systems and SME Operating OS solutions that systemize real daily business workflows.

中文核心定位：

> Optimaks 不做通用 SaaS。我們為特定產業打造可落地的營運 OS，從接單、報價、排程、施工、收款到客戶回流，協助 SME 把日常流程系統化。
