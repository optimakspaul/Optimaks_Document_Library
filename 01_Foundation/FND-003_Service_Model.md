## Service Model

Optimaks uses a productized service model:

1. Setup fee for initial implementation.
2. Monthly fee for hosting, maintenance, updates, support, and reporting.
3. Add-on pricing for larger modules or workflow extensions.
4. Change Request for scope expansion.

## Monthly Fee Scope

Monthly maintenance may include:

- Hosting / deployment maintenance
- Basic bug fixes
- Basic security updates
- Basic backup setup
- Basic uptime monitoring
- 1 small content update per month
- Email / WhatsApp basic support
- Data export assistance

## Cancellation Handover

If a customer cancels monthly service, Optimaks should be able to provide:

1. Static website files or repository access, depending on contract.
2. Customer data export.
3. Domain / DNS / hosting handover notes.
4. Final backup.
5. Handover note.

## Customer Value Proof

Monthly service value should be shown through a maintenance or health report. This is why the Customer Health Report is part of the Optimaks OS roadmap.
