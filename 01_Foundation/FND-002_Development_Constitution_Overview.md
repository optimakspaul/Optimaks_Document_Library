## Document Library Principle

The official source of truth should be Markdown files in GitHub.

```text
Markdown 分檔 = 正式原始文件
Word / PDF = 對外輸出或備份版本
GitHub diff = 版本比對證據
```

## Highest Operating Principle

Any change should be traceable from idea to Change Request, GitHub Issue, branch, implementation, test, merge, deployment, and documentation update.
