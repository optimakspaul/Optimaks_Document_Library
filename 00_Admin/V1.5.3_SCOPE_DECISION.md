## 1. Decision Summary

Optimaks Development Constitution v1.5.3 introduces a formal Research Intake and Multi-Model Handoff layer for AI-assisted development.

This patch does **not** replace v1.5.2 AI GitHub Governance. It strengthens it by defining when and how Gemini may be used as a standalone research and large-context digest tool before ChatGPT, Codex, Antigravity, VS Code, and GitHub execution.

---

## 2. Why This Patch Exists

Optimaks increasingly works with:

```text
- public sector open data
- long API documentation
- competitor research
- client SOP documents
- multi-source market research
- long codebase review
- external SaaS integration documents
```

These inputs are too large or too messy to send directly into implementation agents.

v1.5.3 therefore introduces a controlled intake process:

```text
Large / messy external information
↓
Gemini Research Digest, when useful
↓
ChatGPT Architecture / Strategy Review
↓
AI Context Package
↓
GitHub Issue
↓
Codex / Antigravity implementation
↓
VS Code + GitHub review
```

---

## 3. Core Decision

Gemini may be used as an optional standalone research intake tool when the task involves large context, multiple sources, API documents, public datasets, competitor research, or client SOP files.

Gemini output is **not** a source of truth and must not be sent directly to Codex or Antigravity for implementation.

Gemini output must first be converted into a Research Digest, then reviewed by ChatGPT before it becomes an AI Context Package or GitHub Issue.

## 5. Non-Goals

This patch does not:

```text
- make Gemini mandatory for every issue
- allow Gemini to decide architecture alone
- allow any AI tool to bypass GitHub Issues or PR review
- allow secrets or client sensitive data to be uploaded to AI tools
- replace Codex or Antigravity roles
- replace VS Code / GitHub human review
```

---

## 6. Approved Operating Rule

```text
Gemini = read and digest large information
ChatGPT = review, decide, structure, govern
Codex = implement backend / logic / data / tests
Antigravity = implement UI / app shell / browser verification
VS Code + GitHub = final control, evidence, review, merge
```
