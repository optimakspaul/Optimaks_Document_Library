```text
Fuse the repo/workspace execution, interruption recovery, and release safety governance layers into the official v1.5.8 constitution release.

This update keeps v1.5.7 Short Video Lead Kit Partner Governance intact while adding the daily AI construction safety layer needed for Codex, Antigravity, ChatGPT, Gemini, GitHub, Vercel and Supabase workflows.

Added governance areas:
- GitHub repo as central source of truth
- Local workspace as active construction site
- Full repo context with issue-scoped modification
- One issue / one branch / controlled diff workflow
- Interruption recovery and WIP preservation
- Issue progress log and continuation prompt
- Failed issue recovery and rollback
- Merge readiness gate
- Testing level and validation standard
- Issue size and splitting standard
- Secrets / ENV / API key governance
- Mock / demo / client data separation
- Preview / staging / production deployment gate
- Database migration and RLS change control

This release prevents stale ZIP-based development, uncontrolled AI refactoring, lost WIP state, unsafe merges, secrets exposure, data confusion, and deployment without founder review.
```
