# 16 Partner Governance

Partner governance documents for Optimaks v1.5.7.

This folder defines how external execution partners are evaluated, contacted, managed, and constrained.

External partners are execution/support layers. They are not strategy owners, product owners, client owners, or final quality gatekeepers.
