## Suggested Milestone Structure

| Milestone | Deliverable | Payment / Condition |
|---|---|---|
| M1 Kickoff | Discovery, scope, basic blueprint | Deposit / kickoff payment |
| M2 Preview | Landing page + basic CRM preview | Midpoint payment |
| M3 Acceptance | QA fixes and client acceptance | Pre-launch payment |
| M4 Production | Production deployment and handover | Final payment |
| M5 Maintenance | Monthly maintenance service | Monthly fee |

## Rules

1. Milestones should match clear deliverables.
2. Scope expansion requires Change Request.
3. Payment terms should be written in quotation and agreement.
4. Handover should not happen without acceptance and final payment terms being satisfied.
