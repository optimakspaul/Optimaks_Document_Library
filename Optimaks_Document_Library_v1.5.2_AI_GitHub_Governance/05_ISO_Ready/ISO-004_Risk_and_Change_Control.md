## High-Risk Changes

Treat the following as high-risk:

1. Database schema / migration
2. Authentication
3. Permission / Supabase RLS
4. Billing / payment
5. Production deployment
6. Customer data export/import
7. Contract/legal terms
8. Major pricing changes

## Required Controls

- Change Request
- Impact analysis
- Branch
- Preview / Staging
- QA checklist
- Rollback consideration
- Documentation update when applicable
