## Agreement Checklist

- [ ] Client legal name confirmed
- [ ] Project name confirmed
- [ ] Scope of work attached
- [ ] Deliverables listed
- [ ] Exclusions listed
- [ ] Payment terms stated
- [ ] Monthly service terms stated
- [ ] Cancellation terms stated
- [ ] Data ownership stated
- [ ] Source code / repository ownership stated
- [ ] PDPA / privacy notice included
- [ ] E-sign method confirmed
- [ ] Project start condition stated
