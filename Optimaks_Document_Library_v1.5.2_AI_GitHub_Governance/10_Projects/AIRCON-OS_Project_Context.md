## Purpose

Aircon OS is the first verticalized SME system under Optimaks, focused on aircon service companies.

## Core Modules

1. Landing Page Builder
2. Lead Capture CRM
3. Quote Builder
4. WhatsApp Toolkit
5. Invoice Tracker
6. Booking Schedule
7. Job Card
8. Customer Profile
9. Maintenance Reminder
10. Monthly Health Report

## Plan Tiers

- Starter: website + lead capture + WhatsApp CTA + basic CRM.
- Problem: quotation, invoice, booking and follow-up.
- Growth: automation, maintenance reminder, reporting and client portal.

## Development Principles

1. Starter can be pitched before every future module is complete.
2. Proposal must clearly state included functions and roadmap/add-ons.
3. Demo can show value before full backend is complete.
4. Full automation code should not be handed over casually.
5. Advanced client requests should become Change Request / Add-on Quote.
