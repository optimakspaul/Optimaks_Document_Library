## Purpose

Optimaks OS is the internal operating system for Optimaks. It may include CRM, automation, proposal, quotation, contract, maintenance report, client portal, and project tracking.

## Core Modules

1. CRM Customer Profile
2. Lead Management
3. Proposal / Quotation
4. Invoice / Payment
5. Contract / E-sign
6. Maintenance Report
7. Client Portal
8. Admin Dashboard
9. Settings / Permission
10. Automation

## Development Rule

Optimaks OS is a core system. Changes should be tracked through Change Request and GitHub Issue, especially when they affect database, auth, permission, reporting or client data.
