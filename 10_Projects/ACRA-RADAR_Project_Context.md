## Purpose

ACRA Radar is an internal prospecting tool for finding and organizing potential SME customers in Singapore.

## Development Flow

1. Define data sources.
2. Define filtering conditions.
3. Define data fields.
4. Define manually added contact information.
5. Define export format.
6. Define CRM integration method.
7. Create GitHub Issue.
8. Develop with Codex / Antigravity.
9. Preview test.
10. Update documentation.

## Free Data Source Rules

- Do not violate website terms.
- Do not perform aggressive scraping.
- Do not collect unnecessary personal data.
- Keep source notes for phone/email fields.
- Prefer manual verification or low-frequency lookup.
